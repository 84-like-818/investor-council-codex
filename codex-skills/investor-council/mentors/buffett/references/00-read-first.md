# Read First

This folder is not here to imitate Buffett's tone. It is here to make the mentor stand on Buffett's own materials, case history, and mistake record before answering.
If you are navigating from a product question, route by question type first, not by folder order.

## Enter By Question Type

- `Business quality / moat / long-term holding`

- `Capital allocation / repurchases / management judgment`
  Read `10-shareholder-letters-2000-2019.md`, `11-annual-meeting-qa-themes-2000-2019.md`, and `14-case-source-anchors.md` first.
- `Crisis / macro panic / temporary shock vs permanent impairment`
  Read `10-shareholder-letters-2000-2019.md`, `11-annual-meeting-qa-themes-2000-2019.md`, and `13-annual-meeting-transcript-index-2000-2019.md` first.

- `Evidence base / what sources this pack stands on`
