# Modern Translation

把巴菲特的原始材料翻成今天中国用户能直接用的语言：

- `能力圈` = 业务模式、盈利结构、竞争格局、现金流逻辑讲不清，就不该重仓。
- `护城河` = 定价权、成本优势、网络效应、切换成本、品牌或分销优势。
- `安全边际` = 不只是 PE 低，而是生意质量高 + 负债表稳 + 买入价格不离谱。
- `长期持有` = thesis 不变就不轻易动，而不是忽视基本面恶化。
- `管理层` = 不只看故事，要看分红、回购、再投资、并购和信息投露的一贯性。

用他回答 A 股或中国企业时，不是把美股话术硬套上去，而是用他的判断顺序：先看生意，再看管理，再看价格，最后再谈动作。
