# 投资大师智能团 快速开始

## 默认使用路径

面向普通用户的默认路径只有一条：

1. 安装并登录 Codex Windows app
2. 前往 GitHub Releases 下载 `投资大师智能团-Setup.exe`
3. 双击安装
4. 安装完成后，双击桌面上的“投资大师智能团”
5. 在产品首页选择人物、填写市场背景与问题，然后开始对话

## 你需要提前准备什么

- Windows 11 x64
- 你自己的 Codex / ChatGPT 权限
- 已安装并登录的 Codex Windows app

如果缺少其中任一前提，产品会在首页给出中文阻塞提示，而不是静默失败。

## 安装包会自动帮你做什么

首次安装或首次启动时，产品会自动检查并处理这些事项：
- 检查 Codex 是否已安装
- 检查 Codex 是否已登录
- 同步 `investor-council` skill
- 创建桌面图标
- 检查本地状态目录是否可写
- 检查实时市场数据是否可用
- 检查独立窗口运行环境是否就绪

## 备用路径

- `投资大师智能团-Portable.zip`
  - 适合高级用户或特殊机器环境
  - 不是纯小白默认路径
- `Codex 一句 prompt` 自举
  - 只适合已经安装并登录 Codex、且已经在 Codex 工作区中打开本仓库的高级用户
  - 不替代普通用户的安装包主路径

## 产品边界

- 这不是独立大模型软件，推理仍然通过用户自己的 Codex 权限完成。
- 如果实时行情接口失败，产品会明确提示“实时数据缺口”，不会伪造实时事实。
- 产品不会承诺收益，也不会把输出描述成投资建议或买卖指令。
