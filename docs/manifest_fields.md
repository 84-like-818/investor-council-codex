# manifest 字段说明

| 字段 | 说明 |
|---|---|
| source_id | 唯一 ID |
| title | 标题 |
| url | 主入口 URL |
| landing_url | 备用落地页 |
| category | 分类，例如 primary / biography / image / bibliography |
| source_type | 页面类型，例如 direct_file / landing_page / catalog / article / search |
| access_mode | direct / landing / preview / restricted / search |
| priority | P1 / P2 / P3 |
| license | rights / public-domain / preview / unknown |
| tags | 逗号分隔标签 |
| expected_formats | 希望拿到的格式，逗号分隔 |
| manual_action | 若拿不到，人工下一步 |
| notes | 备注 |
| enabled | 1/0 |
