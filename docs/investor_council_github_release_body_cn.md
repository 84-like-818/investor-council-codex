# 投资大师智能团 v0.1.0

把传奇投资人的长期资料、关键案例与方法论框架，蒸馏成一个可以每天对话的中文思维成长助手。

## 这不是普通投资问答工具

“投资大师智能团”不是简单的角色扮演，也不是一句话荐股器。

它更像一个以传奇投资人物为入口的认知陪练系统：
- 基于大量公开资料、股东信、年会问答、人物经历与案例材料做结构化蒸馏
- 让用户进入人物专属的连续线程，而不是每次从零开始问答
- 把市场讨论从“结论”推进到“框架、节奏、风险与认知成长”

## 你可以期待什么

- 利弗莫尔：趋势、仓位、时机、失效条件与市场节奏
- 巴菲特：企业质量、资本配置、长期持有、错误定价与错误复盘
- 后续会继续扩展更多传奇人物，形成真正的“投资大师智能团”

## 默认安装路径

- 普通用户：下载 `投资大师智能团-Setup.exe`
- 备用路径：`投资大师智能团-Portable.zip`
- 高级用户：已在 Codex 工作区中时，可使用 `Codex 一句 prompt` 自举

## 使用前提

- Windows 11 x64
- 已安装并登录 Codex Windows app
- 用户自有的 Codex / ChatGPT 权限

## 本版包含

- 独立产品壳窗口
- 人物卡片与人物专属线程交接
- 研究员底座驱动的人物包
- 本地自检 / 一键修复
- 中文快速开始、风险提示、隐私说明与使用前提文档

## 重要提示

- 本产品用于教育、研究和信息整理，不构成投资建议、招揽、代客理财或收益承诺。
- 产品默认不采集远程遥测。
- 产品可能访问公开第三方行情接口，因此实时数据可能延迟、缺失或失败；产品会明确提示数据缺口，不会伪造实时事实。
