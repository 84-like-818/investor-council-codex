# 投资大师智能团 v0.1.0

这是一个 Codex 原生产品壳，不是独立大模型。

## 默认安装路径

- 普通用户：下载 `投资大师智能团-Setup.exe`
- 备用路径：`投资大师智能团-Portable.zip`
- 高级用户：已在 Codex 工作区中时，可使用 `Codex 一句 prompt` 自举

## 使用前提

- Windows 11 x64
- 已安装并登录 Codex Windows app
- 用户自有的 Codex / ChatGPT 权限

## 本版包含

- 独立产品壳窗口

- 本地自检 / 一键修复
