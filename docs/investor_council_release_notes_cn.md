# 投资大师智能团 发布说明

## 产品定位

“投资大师智能团”是一个 Codex 原生的传奇投资人物思维成长助手。

它并不试图把投资变成一句话答案，而是希望把传奇投资人的长期经历、方法论演化、关键案例和研究资料，蒸馏成一个可以持续对话、持续复盘、持续成长的中文产品。

它提供：
- 中文产品界面
- 传奇人物入口与资料路由
- 人物专属线程交接
- 本地状态、自检与修复能力
- 面向长期使用的思维陪练体验

真正的推理与对话能力，来自用户自己的 Codex / ChatGPT 权限。

## 核心价值主张

与普通的资讯、选股、指标类工具不同，“投资大师智能团”更关注：
- 你如何理解市场结构
- 你如何处理仓位与节奏
- 你如何设定失效条件
- 你如何在错误中修正自己的认知
- 你如何借传奇人物的长期框架提升自己的判断质量

它不是替你做决定，而是帮助你长出更成熟的思维方式。

## 风险与隐私摘要

- 本产品仅用于教育、研究与信息整理，不构成投资建议、招揽、收益承诺或代客理财服务。
- 产品默认不采集远程遥测。
- 产品会在本地保存必要状态，例如人物记忆、最近交接记录和状态文件。
- 产品可能访问公开第三方行情接口，因此可能产生对外网络请求。
- 产品不会把 Codex 权限说成产品自带能力，也不会删除仓库外任何非产品文件。

## 已知限制

- 新建人物专属线程仍然依赖 Codex 桌面端当前可访问的 UI 控件；如果自动新建失败，产品会回退为“复制到剪贴板 + 聚焦 Codex + 明确提示手动发送”。
- 实时行情层依赖公开接口；当接口异常时，产品会明确提示数据缺口。
- 产品不会替用户登录 Codex；首次使用仍需用户自己完成登录。
