# 投资大师智能团 发布说明

## 版本定位

本产品是一个 Codex 原生产品壳，而不是独立大模型。

它提供：
- 中文产品界面
- 投资人物入口与资料路由
- 人物专属线程交接
- 本地状态、自检与修复能力

真正的推理与对话能力，来自用户自己的 Codex / ChatGPT 权限。

## 权威分发源

GitHub Releases 是唯一权威发布源。

面向普通用户时，推荐直接引导其从 GitHub Releases 下载正式发布包，而不是让其接触源码仓库、脚本细节或内部辅助交付流程。

## 默认安装路径

默认推荐下载：
- `投资大师智能团-Setup.exe`

备用路径：
- `投资大师智能团-Portable.zip`
- `Codex 一句 prompt` 自举（仅高级用户）

## 默认支持环境

- Windows 11 x64
- 已安装并登录的 Codex Windows app
- 用户自有的 Codex / ChatGPT 权限

## 风险与隐私摘要

- 本产品仅用于教育、研究与信息整理，不构成投资建议、招揽、收益承诺或代客理财服务。
- 产品默认不采集远程遥测。
- 产品会在本地保存必要状态，例如人物记忆、最近交接记录和状态文件。
- 产品可能访问公开第三方行情接口，因此可能产生对外网络请求。
- 产品不会把 Codex 权限说成产品自带能力，也不会删除仓库外任何非产品文件。

## 已知限制

- 新建人物专属线程仍然依赖 Codex 桌面端当前可访问的 UI 控件；如果自动新建失败，产品会回退为“复制到剪贴板 + 聚焦 Codex + 明确提示手动发送”。
- 实时行情层依赖公开接口；当接口异常时，产品会明确提示数据缺口。
- 产品不会替用户登录 Codex；首次使用仍需用户自己完成登录。

## 内部辅助工具说明

仓库中如果保留“客户交付包”相关脚本或目录，它们只作为内部运营/支持辅助工具存在，不是公开发布的默认主路径，也不应替代 GitHub Releases 作为权威分发源。
