# 投资大师智能团 Shell

这是“投资大师智能团”的本地壳应用，也是用户真正看到的产品界面。

它承载的不是普通聊天页，而是一个面向中国用户的传奇投资人物思维成长入口：
- 让用户从人物卡片进入传奇投资人的长期框架
- 把市场背景、仓位、标的和问题，交给该人物自己的连续线程
- 让讨论从“问一句结论”，升级为“长期思维陪练”

它负责：
- 渲染中文桌面界面
- 展示人物卡片、诊断状态、最近线程和最近一次发送内容
- 检查 Codex 安装/登录/skill 同步/市场数据/独立窗口运行时
- 把当前问题交接到对应人物的 Codex 专属线程

它不负责：
- 直接充当大模型后端
- 替代 Codex Windows app
- 独立消费 ChatGPT/Codex 配额

正式发布物通过 GitHub Releases 分发；源码入口只保留给开发和支持使用。
