# 投资大师智能团 隐私说明

## 1. 说明范围

本说明描述“投资大师智能团”当前版本在本地运行时会保存什么、不会默认做什么，以及哪些网络请求来自用户自己的环境。

## 2. 默认不采集远程遥测

当前版本默认不启用远程遥测、远程埋点、行为画像上报或云端分析服务。

在默认配置下，产品不会为了统计目的自动把你的提问、人物记忆、最近交接记录或本地文件上传到产品维护方控制的独立服务器。

## 3. 本地会保存哪些信息

为了让产品正常运行，应用会在本机产品目录保存少量状态文件。默认位置通常为：

- `%LOCALAPPDATA%\InvestorCouncilCodex\`

其中可能包括：
- `status.json`：产品运行状态
- `shell_status.json`：壳应用运行状态
- `planned_interest.json`：你对筹备中人物的关注状态
- `handoff_history.json`：最近一次或最近几次交接摘要
- `memory\<mentor-id>.json`：不同人物各自独立的会话记忆

这些文件的用途是让产品记住最近一次交接结果、人物专属线程、局部问题上下文和用户界面状态。

## 4. Codex 登录与权限

本产品不是内置模型软件。推理、对话能力和相关账户权限来自用户自己的 OpenAI / Codex / ChatGPT 环境。

这意味着：
- 用户需要自行安装并登录 Codex Windows app
- 用户的登录态、账户权限和额度管理属于用户自己的 OpenAI 环境
- 本产品不会代替用户保管其 OpenAI 账户，也不会把产品描述成“自带模型权限”

## 5. 第三方行情与公开网络请求

为了显示实时市场状态或补充市场背景，产品可能访问公开第三方行情接口或公开网页数据源。

因此，使用产品时可能发生与以下类别相关的对外网络请求：
- 指数数据
- 个股数据
- 行业或板块数据
- 公开网页抓取或公开接口访问

这些请求可能让第三方服务看到来自你当前网络环境的常规访问元数据，例如公网 IP、访问时间、请求路径、User-Agent 或接口参数。相关第三方如何处理这些数据，受其各自规则约束。

## 6. 不会默认上传哪些内容

在默认配置下，本产品不会主动上传以下内容到产品维护方自建服务器：
- 你的本地非必要文件
- 整个工作区内容
- 你的完整仓库副本
- 你的私人文档
- 与产品无关的电脑文件

如果某项功能需要访问特定本地文件，应以本地处理或明确用户动作触发为前提。

## 7. 文件系统边界

产品会在自己的目录、Codex skill 目录或桌面快捷方式位置写入必要文件，但不应删除仓库外任何非产品文件，也不应清空与你无关的目录。

## 8. 你的控制权

如果你不希望保留某些本地状态，可以删除产品自己的状态目录，或在产品后续版本中使用清理/重置功能。

但请注意，删除这些状态文件可能导致：
- 人物记忆丢失
- 最近交接记录丢失
- 已关注人物状态丢失
- 首次运行检查重新执行

## 9. 说明更新

随着产品功能变化，本说明可能更新。公开发布时，应以 GitHub Releases 中随版本附带的隐私说明文件为准。
