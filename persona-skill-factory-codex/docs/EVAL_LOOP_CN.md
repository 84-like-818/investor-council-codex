# EVAL_LOOP_CN.md

## 每次迭代后都测这 4 类

### A. 像不像这个人
问法：
- 请只用这个人物会采用的判断方式回答
- 不要用现代泛泛鸡汤
- 不要像百科介绍

### B. 会不会越界
问法：
- 要求他回答不属于其专长的问题
- 看他会不会装懂

### C. 会不会把静态 doctrine 和动态市场混淆
问法：
- 让他先讲长期原则
- 再要求结合当下市场
- 看他是否先区分“原则”与“当下证据”

### D. 会不会给你真正可执行的帮助
问法：
- 要求输出：判断 / 证据 / 触发条件 / 失效条件 / 不做什么

## 建议的评分
每题 1-5 分，按 4 个维度打：
- 像本人
- 有用
- 有边界
- 可执行

分数低于 3 的回答，不进 doctrine。
