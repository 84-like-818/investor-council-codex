# 投资大师智能团 使用前提与合规说明

## 1. 这是一个什么产品

“投资大师智能团”是一个 Codex 原生产品壳。

它的主界面、人物入口、交接逻辑、记忆管理和资料路由由本产品负责；真正的推理与对话能力来自用户自己的 Codex / ChatGPT 权限。

## 2. 为什么必须用户自己登录 Codex

当前产品路线不是“内置独立大模型”，而是“外层产品壳 + 用户自己的 Codex 推理引擎”。

因此，用户必须：
- 自己安装 Codex Windows app
- 自己登录自己的 OpenAI / Codex / ChatGPT 账户
- 使用自己账户下可用的权限与额度

这也是本产品保持边界清晰、权限归属清晰和分发透明的重要前提。

## 3. 为什么不是内置模型

当前版本没有把模型直接内嵌在安装包里，也不会把产品描述成“买了软件就自带大模型”。

这样设计的原因包括：
- 让用户明确知道推理能力来自自己的 Codex 环境
- 避免把产品误描述为独立模型服务
- 让用户能够继续使用自己的登录态、设置与权限
- 让产品聚焦在人物系统、研究资料、交接体验和日常使用效率上

## 4. 默认适用环境

当前首发默认支持：
- Windows 11 x64
- 已安装并登录 Codex Windows app 的机器

如果缺少这些前提，产品应明确阻塞提示，而不是继续在半完成状态下运行。

## 5. 默认分发方式

面向普通用户的默认路径是：
- GitHub Releases 下载正式发布包
- 运行 `投资大师智能团-Setup.exe`
- 首次启动后按产品提示完成检查与修复

`Portable.zip` 仅作为备选路径。`Codex 一句 prompt` 仅作为高级用户辅助路径，不是纯小白默认入口。

## 6. 合规边界

本产品用于研究、学习和信息整理，不应被包装为：
- 内置荐股软件
- 保证盈利工具
- 代客下单系统
- 无需用户自己理解风险的自动投资产品

对外介绍、发布说明、安装说明和应用内文案，都应维持这个边界。
